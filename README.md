# DATABASE REPO 

## TỔNG QUÁT
Repo này sẽ bao gồm 2 nội dụng chính: cách thức **viết câu truy vấn SQL** để thao tác với dữ liệu trên cơ sở dữ liệu và các **quy tắc thiết kế database (database design principles)**. Ngoài ra còn có thêm các bước chi tiết cài đặt SQL Server (đóng vai trò là server sinh ra để lưu trữ và phục vụ dữ liệu) và SQL Server Management Studio (SSMS - client tool công cụ để kết nối và thao tác với SQL Server). 

## CẤU TRÚC REPO
- installation.sqlserver-2022: Hướng dẫn cài đặt SQL Server và SSMS.
- sample-db: Bao gồm các file tạo database mẫu và dữ liệu mẫu để thực hành. 
- README.md: File hướng dẫn tổng quát về repo này.
